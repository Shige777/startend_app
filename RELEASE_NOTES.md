# StartEnd App - リリースノート

## v1.0.0-beta.1 (2025-01-12)

### 🎉 ベータ版リリース

StartEnd Appのベータ版をリリースしました！ソーシャル目標達成をサポートする機能を搭載しています。

### ✨ 新機能

#### 🎯 投稿機能
- **START投稿**: 新しい目標の開始を記録
- **END投稿**: 目標の完了を記録  
- **画像付き投稿**: 進捗を視覚的に共有
- **いいね機能**: 投稿にリアクション

#### 👥 ソーシャル機能
- **フォロー機能**: 他のユーザーをフォロー
- **ユーザー検索**: 部分一致検索で簡単に発見
- **プロフィール表示**: 詳細なユーザー情報
- **相互フォロー確認**: フォロー関係の可視化

#### 🏘️ コミュニティ機能
- **コミュニティ作成**: 興味のあるテーマでコミュニティを作成
- **コミュニティ参加/脱退**: 簡単な参加管理
- **コミュニティ検索**: 部分一致検索でコミュニティを発見
- **メンバー管理**: リーダー機能でメンバーを管理
- **活動統計**: コミュニティ内の活動状況を可視化
- **チャット風投稿**: コミュニティ内でのコミュニケーション

#### 📊 プロフィール・軌跡機能
- **軌跡表示**: 個人の投稿履歴を時系列で表示
- **期間フィルター**: 集中・進行中、過去の投稿を分類表示
- **フォロー/フォロワー一覧**: 人間関係の管理
- **コミュニティ一覧**: 参加コミュニティの確認

#### 🔐 認証・設定
- **Google認証**: 簡単なログイン
- **Apple Sign-In**: iOS向けの認証（iOS）
- **メール認証**: 従来の認証方式
- **プロフィール編集**: アイコン、名前、自己紹介の設定

### 🛠️ 技術改善

#### 🔍 検索機能強化
- **部分一致検索**: 投稿、ユーザー、コミュニティで部分一致検索対応
- **検索パフォーマンス**: 効率的な検索結果表示

#### 🐛 バグ修正
- **コミュニティ参加**: 参加画面が重複表示される問題を修正
- **検索結果**: 検索結果のコミュニティタップ時の動作を改善
- **データ同期**: 参加処理後の不要な再読み込みを削除

#### 🎨 UI/UX改善
- **コミュニティ画像**: 表示サイズを2倍に拡大
- **投稿表示順**: 新しい投稿を下に表示
- **レスポンシブデザイン**: 様々な画面サイズに対応

- **Dart**: 3.5.0
- **Firebase**: 最新版
- **対応プラットフォーム**: iOS, Android, Web
